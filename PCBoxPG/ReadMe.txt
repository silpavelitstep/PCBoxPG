Симулятор сборки ПК.
Используемые компоненты:
БП, ЦП, ГП, планки ОЗУ, видеокарта, устройства с интерфейсом SATA.
Функции:
1. Добавление нового компонента в базу, запись в файл.
2. Симулятор сборки: выбор типа компонента, выбор по имени, проверка ключа (DDR/soket), псевдографическое отображение установленного компонента, удаление компонента.
3. Отображение списка установленных компонентов.
4. Симулятор запуска: происходит проверка наличия необходимых компонентов, соответствия частот, максимального объема ОЗУ, достаточной мощности БП. Можно включить отображение списка проблем либо попытаться догадаться самостоятельно.
Легенда к графике:
.... ящик, пустой системный блок
//// или XXXX или |||| или 0000 компонент не установлен
++++ или 1111 компонент установлен
Подробности:
1. Для добавления компонента в базу нужно выбрать его тип, например видеокарта, указать необходимые параметры (имя и прочее) и согласиться с сохранением. Для SATA устройств необходимо выбрать выбрать подтип SSD/HHD или ROM (DVD-ROM). Добавить новый компонент в базу можно в любой момент работы программы. Удалить компонент из базы с помощью этой программы нельзя. Можно удалить только файл целиком средствами ОС, базу придется создавать заново. Для каждого типа устройств отдельный файл. Все SATA устройства в одном файле.
2. В начале пустой системный блок. Туда можно установить только БП и ГП. Прочие компоненты будут доступны только при наличии ГП. При выборе типа компонента будет отображен весь список. Например, для ОЗУ будут отображены планки DDR3 и DDR4. Для выбора нужно указать полное имя. Однако, если ГП не поддерживает тип ОЗУ выбранной планки, будет сообщено об ошибке, планка не будет добавлена, будет произведен переход в главное меню. То же касается ЦП. Если компонент будет добавлен, это будет отображено графически. Если компонен установлен, то можно его заменить, просто выбрать другой такого же типа. Можно его просто удалить. Если можно установить несколько устройств одного типа, например ОЗУ, то программа добавит планку в первый свободный слот, если все слоты заполнены, предложит очистить последний. При удалении ОЗУ планки будет удалена планка из последнего занятого слота. То же касается SATA. Если на ГП установлены компоненты (ЦП, видео, ОЗУ, SATA), то при удалении ГП они будут также удалены.
3. Перед добавлением в блок компонента или после неудачного запуска можно посмотреть список уже установленных компонентов со всеми подробностями, например максимальный объем ОЗУ. Впрочем, список можно посмотреть в любой момент.
4. Сообщение об ошибках:
ПК не запустился: нет БП.
ПК включился и сразу выключился: малая мощность БП.
Шум вентилятора, звуковая сигнализация, нет изображения: не вставлен ЦП, нет ОЗУ.
Только БИОС: не вставлен SSD/HDD.
Тормозят игры: не установлена дискретная видеокарта.
! Отсутствие CD-ROM не влияет на запуск ПК. Нет никаких сообщений об ошибках.
! Можно установить несколько планок ОЗУ разного объема и производителя, несколько разнотипных SATA устройств.
! Если включить отображение списка проблем, программа подробно укажет, какие компоненты не установлены и прочие проблемы, например, доступна не вся память ОЗУ или нет DVD-ROM, фильмы на дисках вы смотреть не можете.
